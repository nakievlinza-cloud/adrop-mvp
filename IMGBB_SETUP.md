# 📸 ImgBB API Setup Guide

## 🚀 Как получить API ключ ImgBB (БЕСПЛАТНО)

### Шаг 1: Регистрация на ImgBB
1. Перейди на https://imgbb.com/
2. Нажми **"Sign Up"** или **"Зарегистрироваться"**
3. Введи email и пароль (или войди через Google)
4. Подтверди email

### Шаг 2: Получи API ключ
1. После входа перейди на https://api.imgbb.com/
2. Нажми **"Get API Key"** или **"Получить API ключ"**
3. Скопируй твой **API Key** (выглядит примерно так: `12345abcde67890`)

### Шаг 3: Добавь ключ в проект
Открой файл `/Users/username/Downloads/ue22/src/lib/imgbb.ts` и замени:

```typescript
const IMGBB_API_KEY = 'YOUR_IMGBB_API_KEY';
```

на:

```typescript
const IMGBB_API_KEY = 'ТВОЙ_КЛЮЧ_ОТСЮДА';
```

### Шаг 4: Готово! 🎉
Теперь загрузка изображений работает через ImgBB!
