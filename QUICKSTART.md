# 🚀 ADROP Telegram Auth - Быстрый старт

## Что уже работает ✅

- ✅ Токен бота: `YOUR_BOT_TOKEN` (задайте в `.env`)
- ✅ Авторизация через Telegram
- ✅ Автозаполнение: имя, username, аватар
- ✅ Выбор роли (Креатор/Бренд)
- ✅ Бот с приветственным сообщением

## Быстрый запуск (4 шага)

### Шаг 1: Запуск приложения

```bash
# Из корня проекта
npm install
npm run dev
```

Приложение запустится на: `http://localhost:3000`

### Шаг 2: Запуск Telegram auth сервера

```bash
# Откройте НОВЫЙ терминал и запустите:
npm run dev:server
```

Убедитесь, что в `.env` заданы `TELEGRAM_BOT_TOKEN` и данные сервисного аккаунта Firebase.

### Шаг 3: Запуск Telegram бота

```bash
# Откройте НОВЫЙ терминал и запустите:
node telegram-bot.cjs
```

Бот запустится и покажет:
```
🤖 Бот запущен!
✅ Бот готов к работе!
📝 Откройте Telegram и найдите @adrop_auth_bot
```

### Шаг 4: Тестирование

1. **Откройте Telegram** → найдите `@adrop_auth_bot`
2. **Нажмите Start** → бот пришлет приветствие
3. **Нажмите кнопку** "🚀 Авторизоваться на ADROP"
4. **На сайте** нажмите "Войти через Telegram"
5. **Выберите роль**: Креатор или Бренд
6. **Готово!** → Вы авторизованы с автозаполненными данными

## Что автозаполняется?

### Из Telegram берется:
- ✅ **Имя**: `first_name` + `last_name`
- ✅ **Username**: `@username` (если есть)
- ✅ **Аватар**: фото профиля
- ✅ **Telegram ID**: для связи с ботом
- ✅ **Ссылка на TikTok**: `tiktok.com/@username`

### Для креаторов дополнительно:
```javascript
{
  about: "✨ Привет! Я [Имя] - креатор UGC контента",
  platforms: {
    tiktok: "https://tiktok.com/@username" // если есть username
  }
}
```

### Для брендов:
```javascript
{
  companyName: "[Имя]",
  companyDescription: "Компания [Имя]"
}
```

## Возможные проблемы

### Бот не запускается?

```bash
# Проверьте Node.js (нужен v16+)
node --version

# Переустановите зависимости
rm -rf node_modules package-lock.json
npm install
```

### Бот не отвечает?

```bash
# Проверьте логи
# Бот должен показать: "✅ Бот готов к работе!"

# Перезапустите бота
# Ctrl+C, затем: node telegram-bot.cjs
```

### OAuth не работает?

- Проверьте токен в `.env`
- Убедитесь что приложение запущено на `localhost:3000`
- Проверьте консоль браузера (F12)

### Данные не заполняются?

```javascript
// Проверьте консоль браузера - должны быть логи:
console.log('🧪 Mock Telegram auth:', authData);

// Если нет данных - проверьте валидацию в telegramAuth.ts
```
