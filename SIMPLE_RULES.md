# 🚀 Простые правила Firebase Firestore

## Скопируй и вставь это в Firebase Console → Firestore → Rules:

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
```

## ✅ Это простейшие правила которые:

1. ✅ Разрешают чтение и запись **всем авторизованным пользователям**
2. ✅ Работают без ошибок парсинга
3. ✅ Подходят для тестирования и разработки

## 📝 Как применить:

1. Открой https://console.firebase.google.com/
2. Выбери проект → **Firestore Database**
3. Нажми на вкладку **Rules**
4. **УДАЛИ всё** что там есть
5. **Вставь** код выше (4 строки)
6. Нажми **Publish**

## ⚠️ После публикации:

- Подожди 1-2 минуты
- Обнови приложение (F5)
- Попробуй отправить приглашение
- Проверь `/debug-notifications`

## ⚙️ Для продакшна (позже):

Когда будешь готов к релизу, можно настроить более строгие правила. Но для тестирования эти правила идеально подойдут!
